# Changelog

这个页面用来记录每个版本的改动，方便回溯功能与 API 变化。

## 0.1.2

- 新增导出参数：`original`（按当前缩放倍数放大导出像素）
- 新增导出参数：`maxSideLength`（限制导出最长边像素，默认 `3000`，传 `0` 关闭）
